Audio to text

The API supports multiple formats: mp4, m4a, wav and flac, and a file size upto 10 MB. It processes the uploaded file and converts it into text with uncanny precision.

Audio Search

Search the audio file for words and phrases. The API combs through the file and locates the exact time of occurrence of a word or phrase. It supports a file size of upto 10 MB and formats like mp4, m4a, wav and flac.
